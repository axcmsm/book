协同创作中心隐私政策

# 协同创作中心隐私政策

# 前言

（更新日期：2023年04月05日）

协同创作中心尊重并保护您的个人隐私权。为了给您提供更准确、更有个性化的服务，协同创作中心会按照本隐私权政策的规定使用和披露您的个人信息。在使用协同创作中心的产品或服务前，请您务必仔细阅读并透彻理解本政策，其中重点条款我们已用加粗形式提示，请特别关注。除本隐私权政策另有规定外，在未征得您事先许可的情况下，协同创作中心不会将这些信息对外披露或向第三方提供。协同创作中心会不时更新本隐私权政策。 您在同意协同创作中心服务使用协议之时，即视为您已经充分理解并同意本隐私权政策全部内容。本隐私权政策属于协同创作中心使用协议不可分割的一部分。

# 1. 我们收集哪些用户信息

a) 除非匿名浏览协同创作中心网站，否则在您注册协同创作中心帐号时，为了便于为您提供服务和产品并与您取得有效的联系，系统将会收集您的用户名、Email地址、IP地址、客户端UA信息。

b) 在您使用协同创作中心服务，或访问协同创作中心平台网页时，协同创作中心自动接收并记录的您的浏览器和计算机上的信息，包括但不限于您的IP地址、操作日志、浏览器的类型、使用的语言、访问日期和时间、软硬件特征信息及您需求的网页记录等数据；

c) 根据国家政策要求，为了合法合规的向您提供产品或服务，明确合同主体和资质条件，您在使用我们的产品或服务的过程中，需要对您在协同创作中心平台上账户进行实名认证。在进行实名认证时，系统可能会收集这些信息：若为自然人认证，需要您提供本人的真实姓名、身份证号码、手机号；若为企业/法人认证，需要您提供单位的名称、公司地址、统一社会信用代码、税务登记证编号、组织机构代码、营业执照副本照片、单位联系人姓名和电话信息。

d) 在您需要协同创作中心邮寄发票、合同等文书文件时，可能需要您填写收件人、收件地址和联系电话信息。

e) 除此之外，部分产品或服务需要提供额外信息的，我们会另外明确告知您并征得您的同意。

# 2. 如何使用收集的信息

a) 协同创作中心不会向任何无关第三方提供、出售、出租、分享或交易您的个人信息，除非事先得到您的许可，或该第三方和协同创作中心（含协同创作中心关联公司）单独或共同为您提供服务，且在该服务结束后，其将被禁止访问包括其以前能够访问的所有这些资料。

b) 您的账号昵称信息，用于用户识别和标识，无法自行修改。

c) 您的中国境内手机号码，用于提供服务通知提醒，以及在需要实名认证的情况下结合您的姓名、身份证号码通过三要素认证。

d) 您的Email地址，用于产品通知、账户状态提醒服务。在需要电子合同、电子发票场景下的邮件送达。在需要技术支持情况下的交流沟通、以及可能会有的可以明确关闭或退订的商业推广信息。

e) 您在实名认证时提交的相关信息，用于国家政策要求和用户主体身份的识别。实名认证信息除非国家相关部门的明确书面要求，否则我们不会在未经您的明确同意前提下向任何第三方提供。

f) 为服务用户的目的，协同创作中心可能通过使用您的个人信息，向您提供您感兴趣的信息，包括但不限于向您发出产品和服务信息，或者与协同创作中心合作伙伴共享信息以便他们向您发送有关其产品和服务的信息（后者需要您的事先同意）。

# 3. Cookie的使用

a) 在您未拒绝接受cookies的情况下，协同创作中心会在您的计算机上设定或取用cookies，以便您能登录或使用依赖于cookies的协同创作中心平台服务或功能。协同创作中心使用cookies可为您提供更加周到的个性化服务，包括推广服务。

b) 您有权选择接受或拒绝接受cookies。您可以通过修改浏览器设置的方式拒绝接受cookies。但如果您选择拒绝接受cookies，则您可能无法登录或使用依赖于cookies的协同创作中心网络服务或功能。

c) 通过协同创作中心所设cookies所取得的有关信息，将适用本政策。

# 4. 我们如何保护、存储、删除您的用户信息

a) 协同创作中心帐号均有安全保护功能，请妥善保管您的用户名及密码信息。协同创作中心将通过对用户密码进行加密等安全措施确保您的信息不丢失，不被滥用和变造。我们将尽最大努力使用各种安全技术和程序来严格保护您个人信息的安全。

b) 协同创作中心在中华人民共和国境内收集和产生的个人信息，将存储在中华人民共和国境内。协同创作中心将会按照相关法律法规允许存储的内容范围和期限进行存储，法律规定不可存储的我们不会进行存储。若法律没有明确规定的，我们按照与您签署的各类协议进行处理。

c) 在使用协同创作中心服务时，请您妥善保护自己的个人信息，仅在必要的情形下向他人提供。如您发现自己的个人信息泄密，尤其是协同创作中心用户名及密码发生泄露，请您立即联络协同创作中心客服，以便协同创作中心采取相应措施。

d) 如果您要求我们删除属于您的个人信息，请通过协同创作中心官网公开的联系电话、邮件等方式联系我们，我们会按照您的要求予以删除。在删除之前您如果需要对一些信息或记录进行保存的，您需要自行备份。

# 5. 我们如何共享、转让、公开披露您的用户信息

在如下情况下，协同创作中心将依据您的个人意愿或法律的规定全部或部分的披露您的个人信息：

a) 经您事先同意，向第三方披露；

b) 为提供您所要求的产品和服务，而必须和第三方分享您的个人信息；

c) 根据法律的有关规定，或者行政或司法机构的要求，向第三方或者行政、司法机构披露；

d) 如您出现违反中国有关法律、法规或者协同创作中心服务协议或相关规则的情况，需要向第三方披露；

e) 如您是适格的知识产权投诉人并已提起投诉，应被投诉人要求，向被投诉人披露，以便双方处理可能的权利纠纷；

f) 在协同创作中心平台上创建的某一交易中，如交易任何一方履行或部分履行了交易义务并提出信息披露请求的，协同创作中心有权决定向该用户提供其交易对方的联络方式等必要信息，以促成交易的完成或纠纷的解决。

g) 其它协同创作中心根据法律、法规或者网站政策认为合适的披露。

# 6. 免责声明

就下列相关事宜的发生，协同创作中心不承担任何法律责任:

a) 由于您将用户密码告知他人或与他人共享注册帐户，由此导致的任何个人信息的泄漏，或其他非因协同创作中心原因导致的个人信息的泄漏；

b) 协同创作中心根据法律法规或政府相关政策的要求、配合国家有权机关的调查取证提供您的个人信息；

c) 任何第三方根据协同创作中心各服务条款及声明中所列明的情况使用您的个人信息，由此所产生的纠纷；

d) 任何由于黑客攻击、电脑病毒侵入或政府管制而造成的暂时性网站关闭；

e) 因不可抗力导致的任何后果；

f) 协同创作中心在各服务条款及声明中列明的使用方式或免责情形。

# 7. 更新隐私政策

我们将不时更新本隐私政策以反映我们的服务和用户反馈。我们鼓励您定期查看此页面以了解最新信息。

如果您对我们的隐私政策有任何问题或疑虑，请随时联系我们的客户支持团队。

# 8. 如何联系我们

a) 如有任何疑问、意见或建议，可以通过客户服务邮箱（axcmsm@qq.com）或微信公众号“代码飞快”联系我们。

b) 我们在成功受理您的问题后，会尽快做出妥善的处理并给与答复。
