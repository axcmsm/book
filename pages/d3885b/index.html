Numpy库

# Numpy库简介

主要用于多维数值指向并行计算。

内置并行运算功能，如果有多核，会并行计算，底层使用C语言编写，内部解除了GIL(全局解释器锁),效率高于存python代码。

有N维数值对象Array，实用的线性代数、傅里叶变换和随机生成函数。

安装:pip install numpy

# 维度

查看维度：ndim

查看几行几列：shape 可重新设置维度

# 数组的形状操作

修改形状：

  • reshape：原数组变换发生改变
  • resize：修改数组真身 无返回值

多维转换成一维：

  • flatten：不影响原数组
  • ravel：直接在原数组里改

数组的叠加：

  • vstack：数组按垂直方向叠加。列需一致
  • hstack：数组想水平方向叠加。行需一致
  • concatenate(a,b,axis):可指定方向 0 垂直 1水平

数组切割：

  • vsplit(数据,几份)：垂直方向七个。
  • hsplit(数据,几份)：水平方向切割。
  • split/array_split(数据,几份,axis=?) :手动指定 0 垂直 1水平

转置：行列转换

  • .T
  • transpose()：返回视图，会影响原数组
  • dot()

# 深拷贝浅拷贝

例：

  • 栈内存(内存中的内存，快内存小)

  • 堆内存(内存中的硬盘，慢内存大)

一般存储的数据会存储到堆内存中，因为数据较大

变量存储到栈内存指向堆内存中的数据。

不拷贝：直接赋值，没有拷贝，同一个栈区改了名字

View或浅拷贝：拷贝了堆内存中的数据的指向(拷贝了变量栈内存)，修改值共享

深拷贝：拷贝了栈内存的，也将堆内存数据也拷贝了一份，不共享

copy():深拷贝

# 随机&axis轴

np.random随机模块：

  • seed:种子
  • rand：0到1之间的随机数 可指定行列
  • randn：值满足标准正太分布
  • randint:指定范围生成随机数
  • choice：随机采样
  • shuffle：洗牌

axis轴：

  • 最外面的括号代表着 axis=0，依次往里的括号对应的 axis 的计数就依次加 1

  • 操作：如果指定轴进行操作，对子元素的0，1，2...分别进行相关操作

  • np.delete:直接删除指定轴下的第几个直接子元素。

# 通用函数

一元函数：

函数 描述
np.abs 绝对值
np.sqrt 开根
np.square 平方
np.exp 计算指数(e^x)
np.log，np.log10，np.log2，np.log1p 求以e为底，以10为低，以2为低，以(1+x)为底的对数
np.sign 将数组中的值标签化，大于0的变成1，等于0的变成0，小于0的变成-1
np.ceil 朝着无穷大的方向取整，比如5.1会变成6，-6.3会变成-6
np.floor 朝着负无穷大方向取证，比如5.1会变成5，-6.3会变成-7
np.rint，np.round 返回四舍五入后的值

np.cos，np.cosh，np.sin，np.sinh，np.tan，np.tanh 三角函数
