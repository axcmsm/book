# 计算机网络

# 第一章

# 概念、功能、组成、分类

概念：网络包含计算机网络、除外还有电话网、铁路网、人体内神经系统网络

计算机网络：是一个分散具有独立功能的计算机系统、通过通信设备与线路连接起来、由功能完善软件实现资源共享和信息传递的系统。

计算机网络是互连、自治的计算机集合。

功能：数据通信、资源共享、分布式处理、提高可靠性、负载均衡......

组成：软件、硬件、协议

工作方式：

  • 边缘部分、用户直接使用：C/S方式、P2P方式
  • 核心部分、为边缘部分服务：网络、路由器..

功能组成：

  • 通信子网：实现数据通信：各种传输介质、通信设备、相应的网络协议组成

    网络层(路由器)、数据链路层（交换机、网桥）、物理层（集线器、中继器）

  • 资源子网：实现资源共享、数据处理：实现资源共享的设备和软件集合

    应用层、表示层、会话层

  • 传输层，位于中间是上下俩层的接口

分类：

  • 分布范围：广域网WAN(交换技术)、城域网MAN、局域网WAN(广播技术)、个人区域网PAN
  • 使用划分：公用网、专业网
  • 交换技术：电路交换、报文交换、分租交换
  • 拓扑结构： 总线型、星型、环型、网状型（常用于广域网）
  • 传输技术：广播式网络(共享公共通信信息)、点对点网络（分组存储转发和路由选择机制）

# 标准化工作（扩展）

标准分类：法定标准(权威机构制定的合法标准、OSI)、事实标准(公司产品占据主流、随时间渐形成行业标准、TCP/IP)

RFC：因特网标准的形式：上升因特网的步骤；因特网草案、建议标准、通过、因特网标准

相关组织：ISO、ITU、IEEE、IETE

# 性能指标

速率：数据率、数据传输率、比特率(1/0位)，单位：b/s,kb/s,Mb/s,Gb/s,Tb/s

带宽：计算机网络中表示网络通信线路传输数据的能力。(网络设备所支持的最高速度)，单位：b/s,kb/s,Mb/s,Gb/s,Tb/s

宽带越大、可向链路发送的数据越多(发送速率)、但传播速率是不变的（2*10^8）m/s。，通道变宽了。

吞吐量：单位时间内通过某个网络(信道、接口)的数据量。受网络带宽或网络的额定速率限制。单位：b/s,kb/s,Mb/s

时延：数据从网络的一段到另一端所需的时间(延迟)，单位s

  • 发送(传输)时延：发送分组的第一个比特到该分组最后一个所需时间=数据长度/信道带宽
  • 传播时延：取决于电磁波传播速率和链路长度=信道长度/电磁波在信息通上传播的熟率
  • 排队时延：等待输出、入链可用
  • 处理时延：检错、找出口

时延带宽积：=传播时延迟X宽带、以比特位单位的链路长度、容量

往返时延RTT：从发送数据开始、到接收对方返回的数据共经历的时延（可 ping www.baidu.com 查看时延）

RTT包括（往返传播时延迟=传播时延迟X2,末端处理时间）

利用率：信道利用率=有数据通过时间/(有+无)数据通过时间、网络利用率=信道利用率加权平均值

# 分层结构

结构：

  • 实体：第n层中活动元素称位n层实体、同一层实体叫对等实体。

  • 协议：进行网络中的对等实体数据交换而建立的规则、标准、或约定、称为网络协议。【水平】

    • 语法：规定传输格式
    • 语义：规定所要完成的功能
    • 同步：规定各种操作顺序

  • 接口：上层调用下层服务的人口

  • 服务：下层为相邻上层提供功能调用【垂直】

计算机网络体系结构是计算机网络的各层及其协议的集合

模型：七层OSI参考模型(法定表准)、4层TCP/IP参考模型(事实标准) =>5层体系结构

OSI：七层：（物、链、网）点到点（输、会、示、用）端到端(不同主机内俩个进程的通信，端口号)：中间系统没有上四层

  • 应用层：能和用户交互产生网络流量的程序。例：FTP、SMTP、HTTP
  • 表示层：用于处理俩个通信系统中交换信息的表达方式（语法和语义）例：数据格式的变换、数据加密解密、数据压缩和恢复。
  • 会话层：向表示层实体进程提供建立连接并且在连接上有序地传输数据。例：建立、管理、终止会话，适用传输大文件。
  • 传输层：负责主机中俩个进程的通信，即端到端的通信。传输单位报文段。例：可靠/不可靠传输、差错控制、流量控制、复用分用）
  • 网络层 ：把分组从源端传到目的端，为分组交换网上的不同主机提供通信服务。传输单位是数据包。例：路由选择、流量控制、差错控制、拥塞控制。
  • 数据链路层：把网络层传下来的数据报组装称帧，传输单位是帧。例：成帧、差错控制、流量控制、访问接入控制。
  • 物理层：在物理媒体上实现比特流的透明传输，传输单位是比特。例：定义接口特性，定义传输模式，定义传输速率，比特同步，比特编码。

TCP/IP：五层：应用层(http\ftp\dns)、传输层(tcp\udp)、网络实际层(ip)、网络接口层(ethernet\atm\frame relay)

  • 应用层：支持各种网络应用 例：FTP、SMTP、HTTP
  • 传输层：进程-进程的数据传输 例：TCP、UDP
  • 网络层：源主机到目的主机的数据分组路由与转发 例：IP、ICMP、OSPF等
  • 网络链路层：把网络层传下来的数据报组装层帧 例：Ethernet、PPP
  • 物理层：比特传输

总结：

  • 相同点

    1. 都分层
2. 基于独立的协议栈的概念
3. 可实现异构网络互联

    2

  • 不同点

    1. OSI定义三点：服务、协议、网络
2. OSI先出现，参考模型先于协议发现、不偏向特定协议
3. TCP/IP设计之初考虑异构网互联问题、将IP作为重要层次。

# 第二章 物理层

# 通信基础

物理层概念：物理层解决任何在连接各种计算机的传输媒介上传输数据比特流、而不是指具体的传输媒介。定义标准

  1. 机械特性：定义物理连接的特性。规定物理连接时所采用的规格、接口形状，引线数目、引脚数量和排列情况。
  2. 电气特性：规定传输二进制位时，线路上信号的电压范围、阻抗匹配、传输速率和距离限制等。
  3. 功能特性：指某条线上出现的某一个电平表示何种意义，接口部件的信号线等用途。
  4. 规程特性：定义各条物理线路的工作规程和时序关系。

基础:通信的目的传送消息

  • 数据：传输信息的实体

  • 信号：数据的电气/电磁的表现，是数据传输过程中的存在形式。

    • 数字信号：代表信息的参数取值是离散的
    • 模拟信号：代表信息的参数取值是连续的

  • 信源：产生和发送数据的源头

  • 信宿：参数和发送数据的源头

  • 信道：信号的传输媒介。

通信方式：单工通信、半双工通信、全双工通信

传输方式：串行传输(速度慢、费用低、距离远)、并行传输(速度快、费用高、近距离，适用计算机内部数据传输)

码元：固定时长的信号波形（数字脉冲）、1码元可以携带的多个比特的信息量。可有多进制的码元。

速率：数据的传输速率。可用码元传输速率(波特)和信息传输速率(比特)表示。转换：n bit信息量 m 码元对应的信息传输速率为：m * n bit/s

计算四进制的码元的传输速率和学习传输速率 4s传8000码 => 8000/4=2000Baud, 信：2000*log2 四 = 4000b/s

系统传输的比特流、通常比较信息传输速率。

# 公式

失真：受码元传输速率、信号传输距离、噪声干扰、传输媒体的质量的影响：较小可识别、较大无法识别

  • 现象：码间串扰：接收端收到的信号波形失去了码元之间清晰界限的现象。
  • 信道带宽：是进到能通过的最高频率和最低频率之差

奈氏准则：在理想低通下、为避免码间串扰、极限码元传输速率为2W Baud，W是信道带宽。单位是Hz。 内忧

理想低通下极限数据传输率=2Wlog2 V （b/s） V是几种码元/码元的离散电平数目。 2W是带宽 Hz

香浓定理：在带宽受限且有噪声的信道中，为了不产生误差，信息的数据传输速率有上限值。 外患

信噪比=信号平均功率/噪声的平均功率，记为S/N。并用分贝db作为度量单位。

信道极限数据传输速率=Wlog2(1+S/N) (b/s) ：S/N为信噪比 W带宽

# 编码调制

信道上传送的信号：基带传输(数字信道)、宽带信号(模拟信道)。传输较近采用基带传输、传输较远宽带传输

编码：数据=>数字信号）：数字数据（数字发送器>数字信号 、模拟数据(pcm编码器 (抽样、量化、编码)）=>数字信号

调制：数据=>模拟信号）：数字数据（调制器调制器>模拟信号、模拟数据(放大调制器）=>模拟信号

数字数据转数字信号 的编码：非归零编码、曼切斯特编码、差分曼彻斯编码、归零、反向不归零、4B、5B编码

数字数据转模拟信号 的调制：调幅、调频、调相、条幅+调相

# 传输介质

概述：数据传输系统中发送设备和接收设备之间的物理通路

分类：导向性传输介质(铜线、光纤)、非导向性传输介质（空气、真空、海水）

导向性传输介质：

  • 双绞线：屏蔽双绞线、非屏蔽双绞线
  • 同轴电缆
  • 光纤：传递光脉冲进行通信，多模(反弹传输)、单模(横向传输)

非导向性传输介质：

  • 无线电波：所有方向传播、有较强的穿透力、远距离、例手机通信
  • 微波：固定方向传播：通信频率较高、频段范围宽、数据率高、例：地面微波接力通信、卫星通信
  • 红外线、激光：固定方向传播：要分别转换为各自的信号格式，即红外光信号和激光信号，再空间中传播。

# 物理层设备

中继器：对信号进行再生和还原。对衰减的信号进行放大。再生数字信号。

俩端：适用于完全相同的俩类物理互联、且俩个网段速率要相同。

543规则：对延迟范围有要求，否则会网络故障。

集线器：多口中继器。再生，放大信号。
