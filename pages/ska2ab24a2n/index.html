scala笔记

# 概述⛳

scala是基于jvm,和java完全兼容,继承了java的很多优点，其类库都能直接调用。

比java更加面向对象,是一门函数式编程语言,静态类高级编程语言,类似于java的加强版。

总结：是一种多范式编程语言

更适合大数据处理,对集合类型数据处理有先天优势.

官网:https://scala-lang.org/ (opens new window)

一叶障目，不见森林。

一次编写到处使用。

spark是新一代内存级大数据计算框架。(为了学好他开肝)

java和scala都是混合型语言(编译成字节码文件,交给jvm解释执行)

c++是编译行语言,编译后直接执行。

柯里化和闭包:

柯里化：把一个列表的多个参数，编程多个参数列表：书写闭包的一种方式 推荐

闭包：一个函数可以访问到它外部的值,这个函数和他所处的环境称为闭包(将所处的一块打包称一个包)

闭包实际上把需要访问到外部的变量打包保存下来，放置堆内存让其引用

因为在嵌套时外面执行结束了,里面就访问不到了,栈内存,执行完就弹出去了

//例:柯里化 实际上是嵌套

      if (int ==0 && string == ""&& char=='0') false else true
    }
    println(func(0)("")('0'))

递归：函数在方法体内又调用了自身。

数学表达式

尾递归(优化):不会不断压栈,对之前的函数进行弹栈在压栈（直接覆盖栈帧) 栈资源只耗费一份。

定义一个内部函数：对内部函数进行调用即可

例：

  //计算阶层
  def tailFact(n:Int):Int={
    //currRes用于传入上一次计算的值。
    @tailrec //@tailrec检验尾递归是否正确
    def loop(n:Int,currRes:Int): Int ={
      if(n==0) return currRes //返回之前全部的计算结果
      loop(n-1,currRes * n) //减一调用  传入计算结果
    }
    loop(n,1)//传入阶层 和 初始值
  }
    println(tailFact(5))

# 集合⭕

三大类：Seq(序列)、set(集)、map(映射) 都有iterable特性

可变:scala.collection.immutable :可直接对原集合进行修改。

不可变:scala.collection.mutable :对象不可修改，每次修改返回一个新对象,类似java中String

提示：操作不不可变用符号,可变用方法，以作区分。

数组：

不可变: (推荐)
	val arr=new Array[Int](10)  //手动 构造方法
	val arr1=Array(12,13,14) //伴生对象调用Array.apply()
	arr(0)=41  //下标访问 =号修改
	//遍历
	for(i <- 0 c arr1.length){print(i)} //until不包含最后一个 下标
	for(i <- arr.indices ){print(i)}
	arr1.foreach(s=>{println(s)}) //lambada
    for (e <- arr1) println(e) // foreach
    //迭代器
    val iter=arr1.iterator;while (iter.hasNext) println(iter.next())
    //连接符打印
    println(arr1.mkString(",")) //推荐
    //添加元素 返回新数组
	//前面添加 val 新数组 = 73 +: 41 +: arr1 :+ 14 //后面添加 		
可变：
	new ArrayBuffer();ArrayBuffer(值....)//伴生对象
	//添加元素 .append() 后 prepend() 前  insert(位置,值1,...)//指定插入 +=:前  +=后
	insertAll(位置,数组) //追加数组
	//删除元素 remove(起始位,删除个数) 不指定个数默认是1    -= 值 对元素进行删除
	//转换数组
	.toArray//不可变  .toBuffer//可变
二维:
	val arr:Array[Array[Int]]=Array.ofDim[Int](外层个数,内层个数)
	arr.foreach(_.foreach(println))//遍历

 //词频统计 每个单词
    val wordCount  =List(
      ("hello",2),
      ("hello world",4),
      ("hi",3),

      .sortWith(_._2>_._2)//对第二个进行倒序排序
      .take(3)//输出前三个
    println(wordCount) //打印

基本语法
val 返回值=变量 match{
	case '变量符合条件' => 执行操作

    case 样例类名("匹配内容") => 执行结果 //对象匹配
	...
	case _ => "通配符"
}
//可以配合op:传代码 使用 默认右break _通配符 不给如果不满足会报错
