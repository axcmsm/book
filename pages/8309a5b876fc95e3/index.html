python简介

# 简介

百度百科 (opens new window)

python官网 (opens new window)

优点： 简单、开源、解析型语言、支持主流范式（面向对象、函数式)、规范程度高，可读性强

缺点：执行效率稍低、代码无法加密、在开发时可以选择的框架太多

应用领域：Web应用后端开发、云基础设施建设、DevOps、网络数据采集（爬虫）、自动化测试、数据分析、机器学习等

解析器

官方的Python解释器是用C语言实现的，也是使用最为广泛的Python解释器，通常称之为CPython。除此之外，Python解释器还有Java语言实现的Jython、C#语言实现的IronPython以及PyPy、Brython、Pyston等版本

# Linux环境

Linux环境自带了Python 2.x版本，但是如果要更新到3.x的版本,可到python官网 (opens new window)下载Python的源代码并通过源代码构建安装的方式进行安装，具体的步骤如下所示（以CentOS为例.

  1. 安装依赖库（因为没有这些依赖库可能在源代码构件安装时因为缺失底层依赖库而失败）。

    yum -y install wget gcc zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel libffi-devel
    1

  2. 下载Python源代码并解压缩到指定目录.

    wget https://www.python.org/ftp/python/3.7.6/Python-3.7.6.tar.xz
xz -d Python-3.7.6.tar.xz
tar -xvf Python-3.7.6.tar
    1
    2
    3

  3. 切换至Python源代码目录并执行下面的命令进行配置和安装。

    cd Python-3.7.6
./configure --prefix=/usr/local/python37 --enable-optimizations
make && make install
    1
    2
    3

  4. 修改用户主目录下名为.bash_profile的文件，配置PATH环境变量并使其生效。

    cd ~
vim .bash_profile
    1

    # ... 此处省略上面的代码 ...

export PATH=$PATH:/usr/local/python37/bin

# ... 此处省略下面的代码 ...
    1
    2
    3
    4
    5

  5. 激活环境变量。

    source .bash_profile

# PEP 8 规范

  1. 单⾏注释#后边应该有⼀个空格
  2. 代码⽂件的最后⼀⾏是空⾏
  3. ⾏内注释需要两个空格（代码和注释之间）

# 管理依赖包

pip指令

  • pip install 包名 #下载
  • pip install 包名 -i 国内源地址 #下载
  • pip uninstall 包名 # 删除包
  • pip list #显示已安装的包
  • pip freeze #显示已安装的包，指定格式显示

可以安装到任意盘符为了方便管理最好要设置同一,安装到配置好环境变量的Scripts里面 提示：安装前找好盘符,例如D盘的python下面的Scripts里面方便日后维护

# anaconda 蟒蛇

机器学习 (opens new window) 机器学习环境配置 (opens new window)

机器学习之前，要安装5个库：scipy、numpy、matplotlib、pandas、sklearn 主要是用来进行:数值运算、计算、绘图、数据分析以及数据挖掘的库

  1. Python自带的IDE —— IDLE，优点是小巧，缺点是需要另外安装很多Python包
  2. Anaconda:已包含很多Python第三方包,缺点:较大且偶尔会崩掉,可以结合PyCharm使用（稳定丰富）pycharm使用Anaconda的解析器
  3. PyCharm：优点是比较稳定，功能强，适合开发大型项目
  4. Jupyter Notebook：在一个文件中方便地保留代码编辑和运行结果，适合分享有很好的文本编辑功能，与很多应用的结合性好；

PyCharm中使用Anaconda Python解释器 (opens new window)

Anaconda安装教程 (opens new window)

jupyter文档 (opens new window)

jupyter 小工具 推荐 效率翻倍🐄 (opens new window) jupyter技巧 (opens new window) 为什么要使用蟒蛇？

它是python的一个集成管理工具或系统，已经把python做相关数据计算与分析所需要的包都集成在了一起，我们只需要安装Anaconda软件就行了，其它什么都不用装，包括python软件,默认安装了python、IPython、Jupyter notebook和集成开发环境 Spyder等等。

总结：安装anaconda让我们省去了大量下载模块包的时间，更加方便 多版本切换

# 基于 python3.6 创建一个名为test_py3 的环境
conda create --name test_py3 python=3.6
# 基于 python2.7 创建一个名为test_py2 的环境
conda create --name test_py2 python=2.7
# 激活 test 环境
activate test_py2 # windows
source activate test_py2 # linux/mac
# 切换到python3
activate test_py3
