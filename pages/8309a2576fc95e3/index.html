# 语言

变量：变量是一种存储数据的载体

命名:见面知义、数字字母下划线组成、数字不能开头、大小写敏感、不跟关键字

类型：整型、浮点型、字符串型、布尔型、复数型

注释："" '' """"

常用函数：

  • int()、eval()：将一个数值或字符串转换成整数，可以指定进制。
  • float()：将一个字符串转换成浮点数。
  • str()：将指定的对象转换成字符串形式，可以指定编码。
  • chr()：将整数转换成该编码对应的字符串（一个字符）。
  • ord()：将字符串（一个字符）转换成对应的编码（整数）。
  • type(): 检查变量的类型
  • input()函数获取键盘输入(字符串)
  • print()函数输出带占位符的字符串 end="\t" 空格 end="\n" 换行
  • encode() 设置编码格式
  • decode() 解码
  • format() 格式化
  • max()、min()、hex()....

格式化用%来实现

例：

占位符 替换内容
%d 整数
%f 浮点数
%s 字符串
%x 十六进制整数

注：（因为百分号代表了占位符，所以带占位符的字符串中要表示百分号必须写成%%）,\表示转义符和换行输入,逗号拼接字符串、r“\n”可让转义功能消失,b”hello“表示按字节输出。

注意：通常在文件开头# -*- coding: utf-8 -*- 来声明编码格式

例：

a = int(input('a = '))
b = int(input('b = '))
print('%d * %d = %.1f' % (a, b, a * b))

break表示跳出循环 continue进入下一次循环

for-in循环:

例：

# 求1~100的求和
sum=0
for x in range(101):
 sum+=x

range():用作构造某个范围：

  • range(101)：可以用来产生0到100范围的整数，需要注意的是取不到101。
  • range(1, 101)：可以用来产生1到100范围的整数，相当于前面是闭区间后面是开区间。
  • range(1, 101, 2)：可以用来产生1到100的奇数，其中2是步长，即每次数值递增的值。
  • range(100, 0, -2)：可以用来产生100到1的偶数，其中-2是步长，即每次数字递减的值。

else: for 循环代码运⾏结束,但是不是被 break 终⽌的时候会执⾏

# 第一次7 11 玩家胜  2 3 12 庄家胜   其他继续
# 第二次7   庄家胜  其他继续
from random import randint
sum=1000
while sum>0:
    print("你的总资产为：",sum)
    needs_go_on=False
    while True:
        debt=int(input("请下注:"))
        if 0<debt<=sum:
            break
    first=randint(1,6)+randint(1,6)
    print("玩家摇出了%d点"%first)
    if first==7 or first==11:
        print("玩家胜")
        sum+=debt
    elif first == 2 or first == 3 or first == 12:
        print("庄家胜")
        sum -= debt
    else:
        needs_go_on = True
    while needs_go_on:# 第二次
        needs_go_on = False
        current=randint(1,6)+randint(1,6)
        print('玩家摇出了%d点' % current)
        if current==7:
            print("庄家胜")
            sum -= debt
        elif current == first:
            print("玩家胜")
            sum += debt
        else:
            needs_go_on=True
print("游戏结束！")

6.完美数 如：6（$6=1+2+3$）和28（$28=1+2+4+7+14$）就是完美数
import math

for num in range(1, 10000):
    result = 0
    for factor in range(1, int(math.sqrt(num)) + 1):
        if num % factor == 0:
            result += factor
            if factor > 1 and num // factor != factor:
                result += num // factor
    if result == num:
        print(num)

7.素数 (只能被1和自身整除的正整数（不包括1）)
import math

for num in range(2, 100):
    is_prime = True
    for factor in range(2, int(math.sqrt(num)) + 1):
        if num % factor == 0:
            is_prime = False
            break
    if is_prime:
        print(num, end=' ')
