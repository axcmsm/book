# MaxWell

概述：

用Java编写的MySQL变更数据抓取软件。它会实时监控Mysql数据库( 二进制日志（Binlog）)的数据变更操作（包括insert、update、delete），并将变更数据以 JSON 格式发送给 Kafka、Kinesi等流数据处理平台。

字段 解释
database 变更数据所属的数据库
table 表更数据所属的表
*type* 数据变更类型
*ts* 数据变更发生的时间
xid 事务id
commit 事务提交标志，可用于重新组装事务
*data* 对于insert类型，表示插入的数据；对于update类型，标识修改之后的数据；对于delete类型，表示删除的数据
*old* 对于update类型，表示修改之前的数据，只包含变更字段

官网地址：http://maxwells-daemon.io/

# 安装

  1. 上传解压安装包|改名

  2. 启动Mysql的binlog

    sudo vim /etc/my.cnf

#添加
[mysqld]

#数据库id
server-id = 1
#启动binlog，该参数的值会作为binlog的文件名
log-bin=mysql-bin
#binlog类型，maxwell要求为row类型
binlog_format=row
#启用binlog的数据库，需根据实际情况作出修改
binlog-do-db=gmall
#binlog-ignore-db=不监控那个数据库,其他都监控



#重启
sudo systemctl restart mysqld

#进入mysql查看 binlog信息
show master status;

  6. 脚本：mxw.sh

    #!/bin/bash

MAXWELL_HOME=/opt/module/maxwell

status_maxwell(){
    result=`ps -ef | grep com.zendesk.maxwell.Maxwell | grep -v grep | wc -l`
    return $result
}


start_maxwell(){
    status_maxwell
    if [[ $? -lt 1 ]]; then
        echo "启动Maxwell"
        $MAXWELL_HOME/bin/maxwell --config $MAXWELL_HOME/config.properties --daemon
    else
        echo "Maxwell正在运行"
    fi
}


stop_maxwell(){
    status_maxwell
    if [[ $? -gt 0 ]]; then
        echo "停止Maxwell"
        ps -ef | grep com.zendesk.maxwell.Maxwell | grep -v grep | awk '{print $2}' | xargs kill -9
    else
        echo "Maxwell未在运行"
    fi
}


case $1 in
    start )
        start_maxwell
    ;;
    stop )
        stop_maxwell
    ;;
    restart )
       stop_maxwell
       start_maxwell
    ;;
esac

# 删除binlog日志

注意：删除操作及其危险，可能会导致数据无法恢复，只能在测试中使用，或者以及确定做好了备份和不在需要使用了，才可以执行此操作。

  1. 连接到MySQL服务器：

    mysql -u username -p
    1

  2. 查看当前所有的二进制日志文件：

    SHOW BINARY LOGS;
    1

    执行此命令后，会列出所有的二进制日志文件的名称和位置。

  3. 删除不需要的二进制日志文件：

    PURGE BINARY LOGS TO 'binlog.000012';
    1

    binlog.000012是你要保留的最新的binlog文件的名称。执行此命令后，MySQL将删除在该文件之前创建的所有二进制日志文件。

    如果你要删除所有的二进制日志，可以使用以下命令：

    PURGE BINARY LOGS BEFORE NOW();
    1

    这将删除所有早于当前时间的二进制日志文件。

  4. 退出MySQL服务器：

    EXIT;
    1

删除binlog会导致，maxwell无法找到上次记录的日志文件：需要到maxwell库中清空，positions表和schemas表。
