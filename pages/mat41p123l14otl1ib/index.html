# matplotlib基本使用

官方文档 (opens new window)

pyplot是核心绘图模块,本质是pyplot中调用matplotlib，顶层类对象Figure(画板),每个画板有每一个绘图对象Axes,相当于画板内的各个子图.各种图表和图例坐标轴是axes的内部元素。

pylab是集大成者。产生常规矩阵运算、线性代数、随机数、FFT等）、绘图功能等等。

例：pylab
from matplotlib.pylab import *
import matplotlib.pylab as pib
print("创建矩阵:",pib.array([1,2],dtype=int)) #调用numpy全局接口
print("生成随机数:",pib.random())#调用numpy.random模块
pib.plot(range(10))#调用了plot

创建画板:

  • plt.figure
  • plt.axes
  • plt.subplot
  • plt.subplots

绘制图表常用

  • plot :Line2D图表接口
  • scatter : 散点图
  • bar/barh :条形图或柱状图
  • hist :直方图
  • pie :饼图
  • imshow:显示图像

配置图例:

  • title，设置图表标题
  • xis/xlim/ylim，设置相应坐标轴范围
  • grid，添加图表网格线
  • legend，在图表中添加label图例参数后，通过legend进行显示
  • xlabel/ylabel，分别用于设置x、y轴标题
  • xticks/yticks，分别用于自定义坐标轴刻度显示
  • text/arrow/annotation，分别在图例指定位置添加文字、箭头和标记，一般很少用
plt.axes()——fig.add_axes()
plt.subplot()——fig.add_subplot()
plt.GridSped()——fig.add_gridspec()

plt.ylabel()——axes.set_ylabel()
plt.xlim()——axes.set_xlim()
plt.ylim()——axes.set_ylim()
plt.title()——axes.set_title()
axes.set()

走向3D：

  • contour伪3D

matplotlib专用3D绘图库：mpl_toolkits

  • plot3D:3D版plot，可用于绘制3维空间的折线图或点图
  • scatter3D:3维散点图
  • bar3D:3维条形图
  • contour3D：3维等高线

更高级的封装：

  • pandas.plot，一个最直接的对matplotlib绘图的封装
  • seaborn，是对matplotlib的高级封装，具有更为美观的图形样式和颜色配置，并提供了常用的统计图形接口，如pairplot()适用于表达多组数据间的关系
  • ggplot，也是对matplotlib进行二次封装的可视化库，主要适用于pandas的DataFrame数据结构

def test7():
    # 获取数据
    addrs = ['南京', '北京', '上海']
    nums = [100, 300, 400]
    # 设置中文
    plt.rcParams['font.family'] = ['SimHei']
    # 设置x轴位置
    x = range(3)
    # 设置标题
    plt.title('2008年私营企业数量')
    # x轴的数据
    plt.xticks(x, addrs)
    # x轴的信息
    plt.xlabel('地区')
    # y轴的信息
    plt.ylabel('户数')
    print(list(zip(x, nums)))
    for m, n in zip(x, nums):
        print("m=" + str(m) + ",n=" + str(n))
        plt.text(m,n , n)
    # 填充数据
    plt.bar(x, nums,width=0.7)
    # 显示图表
    plt.show()
