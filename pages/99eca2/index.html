# pandas概述

[官网](pandas - Python Data Analysis Library (pydata.org) (opens new window))

面板数据 panel data

是强大的分析结构化数据的工具集，基于numpy构建。

  • 提供高性能的矩阵运算
  • 提供大量快速处理函数和方法
  • 应用于数据挖掘和数据分析
  • 提供数据清洗功能

安装：

pip install pandas

# 时间序列

时间序列（time series）数据是一种重要的结构化数据形式。

主要由以下几种:

  • 时间戳（timestamp），特定的时刻。
  • 固定时期（period），如2007年1月或2010年全年。
  • 时间间隔（interval），由起始和结束时间戳表示。时期（period）可以被看做间隔（interval）的特例。

时间和日期数据类型及其工具：

主要会用到datetime、time以及calendar模块。datetime.datetime（也可以简写为datetime）是用得最多的数据类型
