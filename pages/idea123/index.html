Idea 常用快捷键

# java常用快捷键1 🌙

  • alt + insert 重写方法
  • ctrl+h 查看实现类
  • ctrl+alt+b 查看实现类
  • ctrl+f12 查看这个类对于的方法
  • alt + enter 快速导包/类
  • ctrl + p 显示参数信息
  • Ctrl+Alt+T 将代码包在一个块中，例如 try/catch
  • Ctrl + O 重写方法
  • Ctrl + I 实现方法
  • CTRL+ALT+L 格式化代码
  • Shift+F6，重命名
  • Ctrl+N 全局搜素 （查看源码）
  • ctrl+alt+F 快速升级为全局私有变量
  • 双击 SHIFT 在项目的所有目录查找文件
  • ctrl + r 替换快捷键
  • 按下 ins 然后加空格就会往后删除

# scala常用快捷键2 🚗

  1. 快速生成程序入口：main
  2. 自动补全变量：.var
  3. 快速打印：.sout
  4. 快速生成 for 循环：遍历对象.for :输入 1 to 3.for
  5. 查看当前文件的结构：Ctrl + F12
  6. 格式化当前代码：Ctrl + Shift + L
  7. 自动为当前代码补全变量声明：Ctrl + Shift + V
  8. 重写方法 :crtl+o
  9. 显示类结构体(继承) : chrl+h
  10. 代码提示：ctrl + 空格
  11. try补抓异常: chrt+alt+T

# IDEA 其他设置 🚗

  • 设置一张背景图片,路径：File->Settings->Appearance&Behavior->Appearance
  • 滚轮缩放字体大小：在 File -> Setting -> Editor -> General 下进行设置
  • 放大缩小图片大小：在File -> Setting -> Editor -> Images下进行设置
  • 设置编码面板背景颜色 File -> Setting -> Editor ->ColorScheme-> General->text->default text backgroud

# 常见配置问题

  • idea 中 2020 版本执行 main 方法需要添加插件 不然会报错

    File ->Settings -> Build, Execution, Deployment -> Build Tools -> Maven -> Runner 把 Delegate IDE build 取消勾选 这样他就不会全权交给 maven 执行了 或者：参考 (opens new window) 今天又报错了... 是2020版本这个兼容性问题的影响,我换了另一个版本就可以解决。
